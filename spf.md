# `spf`

## Descripción

`spf` (superfile) es un gestor de archivos de terminal moderno y elegante.

## Instalación

### Linux y macOS

```bash
bash -c "$(curl -sLo- https://superfile.dev/install.sh)"
```

### Windows

#### Powershell
```powershell
powershell -ExecutionPolicy Bypass -Command "Invoke-Expression ((New-Object System.Net.WebClient).DownloadString('https://superfile.dev/install.ps1'))"
```

#### Winget
```bash
winget install --id yorukot.superfile
```

#### Scoop
```bash
scoop install superfile
```

## Sintaxis de Uso

```bash
spf
```

## Opciones Válidas

La documentación principal no detalla las opciones de línea de comandos. `spf` se utiliza principalmente de forma interactiva.

## Ejemplos de Uso

*   **Iniciar el gestor de archivos:**
    ```bash
    spf
    ```
    Esto iniciará `spf` en el directorio actual.
